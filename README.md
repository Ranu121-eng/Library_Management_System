# Library Management System

This is a simple Library Management System project developed using
HTML, CSS, Python (Flask), and MySQL.

## Features
- User registration
- Issue and return books
- Book records stored in MySQL database

## Technologies Used
- Frontend: HTML, CSS
- Backend: Python (Flask)
- Database: MySQL

## How to Run
1. Install Python
2. Install required packages
3. Run app.py
4. Open browser and go to localhost
